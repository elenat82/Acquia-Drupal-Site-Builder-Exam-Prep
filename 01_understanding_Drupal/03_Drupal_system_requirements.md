## Requisiti del server web

Il server deve avere una versione di PHP compatibile, per Drupal 10 è consigliata la versione di PHP 8.1.6.

Drupal e PHP devono avere i permessi di lettura e scrittura per la directory /sites/default/files. Quest'area viene utilizzata per memorizzare i file nella cache (CSS e JavaScript compressi) e tutti i file caricati tramite l'interfaccia di Drupal. Non utilizzare permessi tipo chmod 777.
